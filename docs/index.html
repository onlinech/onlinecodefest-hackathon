What have we done...

Time tracking tool

Based on Arduino Uno and ASP.NET MVC we created a tool to measure a working time. It contains two parts - ASP.NET MVC application with the nice user UI and a hardware box. Hardware box measures automatically the time when it is in concrete position. When you put it on the back, then it measures the time defined earlier in UI application e.g. for Komax related work. Then, when you put it in front, it measures the time related to Oerlikon. It has 6 tops which can be defined for different types/codes of activities in the UI application. To start measuring the time, user needs to click Start button in the application.

Time tracking system

Based on React Native and ASP.NET we created a system to simplify the time tracking and reporting it to the Navision. It contains two parts - ASP.NET services and an Android application. In the Android application, user can see all the tasks from the TFS which are assigned to him and ready for work. Then it measures the time spent on the task. While working on a task in application TFS tasks gets updated as well. At the end report can be generated with all informations needed for Navision.

Online Whiteboard

Based on ASP.NET and Javascript we created an Online Whiteboard to simplify drawings and showings during the Skype call or presentation. It contains two parts - Server written in ASP.NET and the client part in HTML5 and Javascript. To communicate between the client and the server we used the SignalR library which supports the Web Socket protocol. Our problem which we have at work is to share ideas and discuss complex things with people who are not in the same room. Usually we discuss many things in front of the Whiteboard but via Skype this is not easily possible. To solve this problem we wanted to build an online whiteboard. Every person which has the given link can connect to the whiteboard and start drawing. The viewers can do following things: draw strokes, rectangles, circles, text, change color, change the username, chat, clear the board, share the board (copies link to the clipboard), create a new board, share the mouse cursor. The whole project is hosted on an Azure WebApp and each time the main branch of the GitHub repository is updated, Azure automatically deploys the newest version.You can find it here: Online Whiteboard

Reserve the room

Based on Arduino and Javascript - NodeJs and ReactJs we wanted to create a system where users could book meeting rooms for Last minute meetings. Since sometimes when we need to talk with clients ”on the fly”, we need to seek a free room to have a short call. The solution, was to create a web page were will indicate which room(s) are available, and a physical button in each room to mark the room as “occupied”.
